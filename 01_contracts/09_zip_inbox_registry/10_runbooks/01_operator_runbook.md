# Operator Runbook

## Steps
1. Download a ZIP package directly into the right project folder in `_zip_inbox/`
2. Do not manually extract it
3. Do not rename it after the canonical naming standard is adopted
4. Run inbox validation
5. Refresh the registry
6. Generate an install plan
7. Review findings
8. Run apply mode
9. Review report, ledger and archive note

## Escalate if
- duplicate sequences exist
- foreign project slugs appear
- a gap blocks the required order
- a manifest is missing and generic defaults are unsafe
