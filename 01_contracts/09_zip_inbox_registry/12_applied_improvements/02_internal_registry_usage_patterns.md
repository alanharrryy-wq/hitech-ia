# Internal Registry Usage Patterns

## Pattern 1: daily scan
Run validation and registry generation across the entire inbox tree.

## Pattern 2: single-project apply
Generate a plan for one project slug and apply it.

## Pattern 3: failure triage
Filter the registry for `failed` or `quarantined` entries and inspect their reports.

## Pattern 4: audit trail
Compare current registry state against prior snapshots to understand what changed.
