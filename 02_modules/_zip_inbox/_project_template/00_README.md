# Project Inbox Template

Copy this folder and rename it to `<project_slug>` when creating a new project inbox queue.

Required ZIP naming:

`zip<sequence>_<project_slug>_<package_slug>.zip`

Minimum contents:
- ZIP files in sequence order
- `project.manifest.json` (optional)
- `_processed/`, `_failed/`, and `_staging/` operational subfolders
