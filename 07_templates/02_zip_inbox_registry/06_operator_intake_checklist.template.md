# Operator Intake Checklist

- [ ] ZIP downloaded directly to the correct project folder
- [ ] Filename starts with `zip<sequence>_`
- [ ] Project slug in filename matches folder name
- [ ] No unrelated project ZIPs in the same folder
- [ ] `project.manifest.json` exists or generic defaults are acceptable
- [ ] No manual extraction performed
- [ ] Validation run completed
- [ ] Registry refreshed
