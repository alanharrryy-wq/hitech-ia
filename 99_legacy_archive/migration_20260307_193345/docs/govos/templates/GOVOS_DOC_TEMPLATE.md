# GOVOS Governance Document Template

## 1. Purpose

Template for all canonical governance documents.

## 2. Scope

Use for new documents in `u1` to `u4` and shared governance docs.

## 3. Authority

Canonical governance docs must conform to this structure unless a documented exception exists.

## 4. Terms

Replace placeholders with deterministic values and explicit ownership.

## 5. Non-Negotiables

- Deterministic language only.
- Explicit acceptance gates.
- Non-negotiables listed in plain text.

## 6. Anti-Patterns

- Vague commitments without validation criteria.
- Optional enforcement language for mandatory controls.

## 7. Controls

Define immutable controls, allowed mutations, and rollback conditions.

## 8. Procedure

List step-by-step execution with deterministic ordering.

## 9. Evidence Outputs

List required report files, schemas, and pass/fail outputs.

## 10. Validation Gates

Define hard blockers, warnings, and pass criteria.

## 11. Exceptions and Escalation

Specify exception authority and required evidence for temporary waivers.

## 12. Change Control

Track version bumps, compatibility notes, and migration instructions.
